Foi ensinado os principais comandos do git, desde a criação de tabelas no terminal até o "push" no GitHub.
Foi passada a importância do: 

"mkdir <folder>" 
// Criação de uma pasta;

"cd <folder>" 
// Entrando na pasta;

"touch <file>" 
// Criando um arquivo;

"git init" 
// Cria um novo repositório do Git;

/*Abrir uma ferramenta de texto ou utilizar o nano direto no terminal para inserção dos dados no arquivo.*/

"git add . (ou) git add <file>"
// Adiciona alteração no diretório ativo à área de staging ;

"git reset head" 
// É um comando usado para desfazer alterações locais no estado de um repositório do Git. 

"git commit -m <file>" 
//Salva a modificação feita no arquivo;

"git checkout -b <nome_da_branch>" 
// Cria uma nova branch e se conecta a ela.

"git branch" 
// Consegue vizualizar quais branchs tem no arquivo.

"git merge <nome_da_branch_que_deseja_juntar>"
// Junta uma branch a linha cronológica principal.

"git remote add origin <link do repositorio>" 
// para adicionar um origin para o repositório se não existe nenhum.

"git push -u origin <nome_da_branch_principal>"
// É executado para efetuar postagem das alterações locais para o repositório online. 

"git clone <repositório_que_deseja_clonar>"
// Permite clonar um repositorio completo de uma pessoa.
